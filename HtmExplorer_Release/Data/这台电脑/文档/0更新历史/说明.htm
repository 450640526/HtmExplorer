说明

 

最低环境

系统安装了.NET 4.0                地址：http://www.baidu.com/s?word=.NET+4.0&tn=82013038_35_hao_pg&ie=utf-8
浏览器的版本最低 IE9.0        否则网页中的图片无法查看

 

文档

默认文档 和 主程序同一目录的我的文件夹下 文档格式为.htm 可以直接在浏览器中打开

 

附件

可以添加文件，也可以添加文件夹，快速添加的方法把文件或者文件夹往附件窗体上拖拽

 

搜索查询

程序会搜索以下符合条件内容（不区分大小写）
文件名称相同
关键字相同
输入* 列出所有的 .htm文档

 

导入导出

支持txt转换成htm

htm,html带_files 携带外链接图片的网页 转换成单个网页 .htm

rtf转换成htm

 

HTM编辑器

  2. 查找        Ctrl+F
  3. 替换        Ctrl+H
  4. 缩放        Ctrl+滚轮
  5. 恢复正常大小        Ctrl+0
